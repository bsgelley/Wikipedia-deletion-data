This archive file contains a set of 847 Wikipedia articles Proposed for Deletion, or PROD'ed, from October-December 2011.
This set also includes 141 articles PROD'ed in March 2013. Folder structure is similar to that of the Speedy set; 
each folder is labeled with the date the articles in it were PROD'ed; for the 2011 data, the format is as follows: PRODs , followed by the date.
For the March 2013 data, each day's folder is labeled with only the date. Each day's folder has a subfolder entitled "Deleted " containing all
pages PRODed on that day that were later deleted. As above, relevant talk pages are included.
